7.1 选择一个行政单位 department-tree.coffee

行政单位选择控件 gdk-api="tree-dropdown" Options:
Name type default description
value string 对应的ID
text string 显示在界面上的文本
url string ajax 请求的 URL
min int 允许选择的最小类型，参考 DepartmentInfo_Type, LocationInfo_Type
max int 允许选择的最大类型，参考 DepartmentInfo_Type, LocationInfo_Type
