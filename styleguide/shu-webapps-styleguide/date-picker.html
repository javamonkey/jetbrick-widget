2.1 单个日历 date-picker.coffee

单个日历 gdk-api="datepicker" Options:
Name type default description
value date 默认值，格式：yyyy-MM-dd
args-required boolean false 是否允许为空
args-min date 允许选择的最小日期
args-max date 允许选择的最大日期

2.2 选择日历范围 date-picker.coffee

日期范围 gdk-api="datepicker-range" Options:
Name type default description
value date ~ date 默认值，格式：yyyy-MM-dd ~ yyyy-MM-dd
args-required boolean false 是否允许为空
args-min date 允许选择的最小日期
args-max date 允许选择的最大日期
args-inputs string, string 将开始日期和结束日期分成2个不同的Input对象。由逗号分割的2个name值组成。
