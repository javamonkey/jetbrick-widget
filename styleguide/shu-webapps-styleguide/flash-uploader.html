3.1 Flash 上传文件 flash-uploader.coffee

flash_upload_create(webroot, upload_url, flash_id, support_types, button_style, completed_callback)
Parameter Name type Optional description
webroot string false webapp 的根目录
upload_url string false 处理上传文件的 URL
flash_id string false 要替换 Flash 的 dom 对象
support_types string false 支持上传的文件类型
button_style string true 显示的按钮样式, 目前只有 "01" 和 "02" 两种，默认 "01".
completed_callback function true 文件上传完成后的回调函数，如果指定，需要自己实现该函数。函数原型：function(fid, data)
